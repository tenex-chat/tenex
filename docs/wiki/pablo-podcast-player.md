# Pablo Podcast Player

## Overview

The Pablo Podcast Player agent (5f6280b5) is a remote agent running on the Podcastr App on iPhone that creates audio podcasts with status briefings for Pablo. [^e2340-3]


## Audio Briefing Workflow

The Chief of Staff periodically delegates audio briefings to the Podcast Player, sending key updates (especially blockers) and always instructing it to "create a podcast" with that information. [^e2340-4]

## Protocol Persistence

The Podcast Player protocol for audio briefings is persisted in the Chief of Staff's standing rules file (`+STANDING-RULES.md`). [^e2340-5]
