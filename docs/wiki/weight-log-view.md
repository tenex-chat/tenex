# Weight Log View

## Weight Log View

The weight log view displays all weight entries most-recent-first, with swipe-to-delete and tap-to-edit interactions. Editing uses +/-0.1 stepper buttons and a Save button. Weight changes require an explicit 'Log Weight'/'Update' button press to save instead of auto-saving after a timeout. [^c6a12-14]
