# RAG Knowledge Management

This skill provides tools for semantic search across project knowledge, creating and managing vector database collections, adding documents, and managing real-time data subscriptions.

## Available Tools
- `rag_search` - Search across indexed project knowledge
- `rag_add_documents` - Add documents to a RAG collection
- `rag_collection_create` - Create a new vector database collection
- `rag_collection_delete` - Delete a collection
- `rag_collection_list` - List all available collections
- `rag_subscription_create` - Create a persistent MCP resource subscription
- `rag_subscription_list` - List all active subscriptions
- `rag_subscription_get` - Get details for a specific subscription
- `rag_subscription_delete` - Delete a subscription
