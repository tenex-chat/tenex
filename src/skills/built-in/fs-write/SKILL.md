# Filesystem Write

This skill provides file writing and editing tools for the project and the agent's home directory.

## Available Tools
- `fs_write` - Write content to a file in the project
- `fs_edit` - Edit a file in the project by replacing a specific string
- `home_fs_write` - Write content to a file in the agent's home directory
- `home_fs_edit` - Edit a file in the agent's home directory by replacing a specific string
